COMMON PARAMETERS - 

These are available on Mod machines and saved per preset

TAPDIV (TAP DIVISIONS):

 When tapping tempo, BPM can be set for Whole, Half, Quarter, Dotted Eight, Eighth, Triplets, or 

Sixteenth notes

PR/PST (PRE/POST):

 If using a pre/post setup, this selects whether the effect in the preset is pre or post.

NAME:

 Allows editing the 16 character name of the current preset. Use the VALUE encoder to change the selected character. 

Use the TYPE encoder to select a character. Exit by pressing the VALUE encoder, then press and hold the TYPE encoder to 

save

 the name permanently. 

Note: 

For the preset name to be displayed the NAMES global must be set to ON or SCROLL.

EP (EXPRESSION PEDAL ON/OFF):

 Configure the Expression pedal input to be ON or OFF.

EP SET (EXPRESSION PEDAL SET UP):

 Set up which knobs are controlled by the expression pedal. Turn the knobs while HEEL 

is displayed to select the minimum setting for each knob to be controlled. Turn the knobs while TOE is displayed to select the 

maximum setting for each knob to be controlled.

MIDI CL (MIDI CLOCK ON/OFF):

 Configure whether Mobius listens for incoming MIDI Clock messages.
